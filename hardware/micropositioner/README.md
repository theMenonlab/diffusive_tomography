# TriAxis Micropositioner Hardware Files

This folder contains the printable micropositioner files used for the CWDT illumination-fiber positioning hardware.

## Contents

- `stl/` - STL exports for the printed parts.
- `print_profile/TriAxis_Print.3mf` - Bambu/3MF print project.

## Notes

The public model description reports a fully printed stage using about 29 g of PLA, approximately 13 mm of travel, and nominal 10 micron adjustment increments using M6 threading. Treat those as design/use specifications rather than independent metrology.
